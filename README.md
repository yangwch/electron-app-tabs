# 开发步骤

## 运行

1. 运行 client （vite）
```bash
  $ cd client
  $ npm install
  $¸ npm run dev
```
2. 运行 server （electron）
```bash
$ cd server
$ npm install
$ npm start

```

## 打包

1. 打包 client （vite）
```bash
$ cd client
$ npm install
# 打包到server/build
$ npm run build
```

2. 打包 server （electron）
```bash
$ cd server
$ npm install
$ npm run make
```
